# 问题2: 均线金叉分析

## 数学逻辑

### 2.1 移动平均线定义

n日移动平均线：
MA(n, t) = (1/n) * ΣP(t-i), i从0到n-1

具体形式：
MA(5, t) = (P(t) + P(t-1) + P(t-2) + P(t-3) + P(t-4)) / 5
MA(10, t) = (P(t) + P(t-1) + P(t-2) + ... + P(t-9)) / 10

### 2.2 金叉定义

金叉(t) = 1, 如果 MA(5, t-1) < MA(10, t-1) 且 MA(5, t) >= MA(10, t)
        = 0, 否则

数学含义：
- 条件1：MA(5, t-1) < MA(10, t-1) 表示前一日短期均线在长期均线下方
- 条件2：MA(5, t) >= MA(10, t) 表示当日短期均线向上穿过长期均线

### 2.3 次日涨跌幅

R(t+1) = (P(t+1) - P(t))/P(t) * 100%

---

## 建模过程

### 步骤1：计算5日均线
对每只股票，按日期排序：
MA5(t) = rolling_mean(P, window=5)

### 步骤2：计算10日均线
MA10(t) = rolling_mean(P, window=10)

### 步骤3：计算次日涨跌幅(全局平移)
R_next(t) = shift(R(t), -1) = (P(t+1) - P(t))/P(t) * 100%

### 步骤4：计算均线前一时刻值
MA5(t-1) = shift(MA5(t), 1)
MA10(t-1) = shift(MA10(t), 1)

### 步骤5：判断金叉
G(t) = I(MA5(t-1) < MA10(t-1) 且 MA5(t) >= MA10(t))

其中I(.)为指示函数

### 步骤6：筛选金叉样本并统计
设金叉样本集为S_G = {t: G(t) = 1}

上涨概率：
P_up = N(t in S_G 且 R_next(t) > 0) / N(S_G) * 100%

平均涨跌幅：
R_mean = (1/N(S_G)) * Σ R_next(t), t属于S_G

标准差：
S = sqrt((1/(N-1)) * Σ (R_next(t) - R_mean)^2)

95%置信区间：
R_mean ± 1.96 * S / sqrt(N)

## 最终结果

| 指标 | 数值 |
|------|------|
| 样本数 | 65,030 |
| 上涨概率 | 47.21% |
| 平均涨跌幅 | 0.1218% |
| 标准差 | 3.13% |
| 95%置信区间 | [0.1005%, 0.1431%] |

---

## 结论

金叉策略无效：
- 上涨概率47.21% < 50%(随机概率)
- 平均涨跌幅0.12%接近于0
- 置信区间虽不含零但数值极小
