# 问题2: 均线金叉分析

## 解题思路

1. **计算均线**：计算5日和10日移动平均线
2. **识别金叉**：当前一日MA5 < MA10，今日MA5 ≥ MA10时，形成金叉
3. **买入假设**：以金叉出现日为买入点
4. **收益计算**：计算买入后次日的涨跌幅

## 数学假设

- MA5 = 5日收盘价均值
- MA10 = 10日收盘价均值
- 金叉条件：MA5_prev < MA10_prev 且 MA5 ≥ MA10

## 建模过程

```python
# 均线计算
ma5 = close.rolling(5).mean()
ma10 = close.rolling(10).mean()

# 金叉判断
golden = (ma5_prev < ma10_prev) & (ma5 >= ma10)
```
