## 问题1：涨停次日走势分析

### 解题思路

1. **涨停识别**：根据A股涨跌幅限制，主板涨幅≥9.9%视为涨停
2. **分组分析**：主板和创业板分别统计
3. **概率计算**：统计涨停后次日各涨幅区间的分布

### 数学模型

- 主板涨停：pct_chg ≥ 9.9%
- 创业板涨停：pct_chg ≥ 9.9%
- 次日涨幅区间分类：
  - 主板：涨停、>7%、5~7%、3~5%、0~3%、-3~0%、-5~-3%、<-5%
  - 创业板：涨停、>10%、7~10%、5~7%、3~5%、0~3%、-3~0%、-5~-3%、-7~-5%、-10~-7%、<-10%

### 核心代码

```python
# 涨停识别

# 获取次日涨跌幅
```

### 结论

## 问题2：均线金叉买入分析

### 解题思路

1. **计算均线**：5日均线(MA5)和10日均线(MA10)
2. **金叉识别**：MA5从下往上穿过MA10
3. **收益计算**：买入后次日的涨跌幅
4. **统计检验**：计算上涨概率、均值、置信区间

### 数学模型

- MA5 = Σclose(i) / 5 (i=0~4)
- MA10 = Σclose(i) / 10 (i=0~9)
- 金叉条件：MA5(t-1) < MA10(t-1) 且 MA5(t) ≥ MA10(t)

### 核心代码

```python
# 均线计算
df['ma5'] = df.groupby('ts_code')['close'].rolling(5).mean()
df['ma10'] = df.groupby('ts_code')['close'].rolling(10).mean()

# 金叉判断
df['golden'] = (df['ma5_prev'] < df['ma10_prev']) & (df['ma5'] >= df['ma10'])
```

## 问题3：金山谷形态分析

### 解题思路

1. **计算均线**：MA5、MA10、MA20
2. **金山谷识别**：MA5上穿MA10和MA20后形成的第二个三角形
3. **买入信号**：金山谷出现日
4. **收益分析**：短期(10日左右)收益

### 数学模型

- MA5 = 5日均价（短期）
- MA10 = 10日均价（中期）
- MA20 = 20日均价（长期）
- 金山谷条件：
  - MA5_prev < MA10_prev AND MA5_prev < MA20_prev
  - MA5 ≥ MA10 AND MA5 ≥ MA20

### 最终结果

| 指标 | 数值 |
|------|------|
| 金山谷数量 | 7,411 |
| 次日分析样本 | 3,502 |
| 上涨概率 | 74.56% |
| 平均涨跌幅 | 2.4345% |
| 95%置信区间 | [2.3027%, 2.5664%] |

### 结论
金山谷是强烈的买入信号，上涨概率75%，平均收益2.4%。
