# 问题3: 金山谷形态分析

## 解题思路

1. **计算均线**：计算5日、10日、20日移动平均线
2. **识别金山谷**：
   - 短期均线(MA5)上穿中期均线(MA10)和长期均线(MA20)
   - 形成尖头向上的三角形
3. **买入信号**：以金山谷出现日为买入点
4. **收益分析**：分析买入后短期内的涨跌幅

## 数学假设

- MA5 = 5日收盘价均值（短期均线）
- MA10 = 10日收盘价均值（中期均线）
- MA20 = 20日收盘价均值（长期均线）
- 金山谷条件：
  - MA5_prev < MA10_prev 且 MA5_prev < MA20_prev
  - MA5 ≥ MA10 且 MA5 ≥ MA20

## 建模过程

```python
# 计算均线
ma5 = close.rolling(5).mean()
ma10 = close.rolling(10).mean()
ma20 = close.rolling(20).mean()

# 金山谷判断
golden_valley = (ma5_prev < ma10_prev) & (ma5_prev < ma20_prev) & 
                (ma5 >= ma10) & (ma5 >= ma20)
```
