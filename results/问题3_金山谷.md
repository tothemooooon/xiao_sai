# 问题3: 金山谷形态分析

## 数学逻辑

### 3.1 均线定义

MA(5, t) = (P(t) + P(t-1) + P(t-2) + P(t-3) + P(t-4)) / 5
MA(10, t) = (P(t) + P(t-1) + ... + P(t-9)) / 10
MA(20, t) = (P(t) + P(t-1) + ... + P(t-19)) / 20

### 3.2 银山谷定义(Silver Valley)

银山谷(t) = 1, 如果 MA(5,t-1) < MA(10,t-1) 且 MA(5,t-1) < MA(20,t-1) 且 MA(5,t) >= MA(10,t) 且 MA(5,t) >= MA(20,t)
           = 0, 否则

数学含义：
- 条件1：MA(5,t-1) < MA(10,t-1) < MA(20,t-1) —— 前一日MA5在MA10和MA20下方
- 条件2：MA(5,t) >= MA(10,t) 且 MA(5,t) >= MA(20,t) —— 当日MA5上穿MA10和MA20
- 这是MA5首次同时上穿MA10和MA20

### 3.3 金山谷定义(Golden Valley)

设银山谷首次出现的时刻为t_silver，则：

金山谷(t) = 1, 如果 存在k>0使得 银山谷(t-k)=1 且 MA(5,t) >= MA(10,t) 且 MA(5,t) >= MA(20,t)
          = 0, 否则

### 3.4 金山谷vs银山谷约束对比

| 形态 | MA5(t-1)与MA10(t-1) | MA5(t-1)与MA20(t-1) | MA5(t)与MA10(t) | MA5(t)与MA20(t) | 时间约束 |
|------|----------------------|----------------------|----------------|----------------|----------|
| 银山谷 | 小于 | 小于 | 大于等于 | 大于等于 | 首次金叉 |
| 金山谷 | 小于 | 小于 | 大于等于 | 大于等于 | 银山谷之后再次金叉 |

金山谷相比银山谷的额外约束：T_gold > T_silver

## 建模过程

### 步骤1：计算5日均线
MA5(t) = rolling_mean(P, window=5)

### 步骤2：计算10日均线
MA10(t) = rolling_mean(P, window=10)

### 步骤3：计算20日均线
MA20(t) = rolling_mean(P, window=20)

### 步骤4：计算次日涨跌幅(全局平移)
R_next(t) = shift(R(t), -1)

### 步骤5：计算均线前一时刻值
MA5(t-1) = shift(MA5(t), 1)
MA10(t-1) = shift(MA10(t), 1)
MA20(t-1) = shift(MA20(t), 1)

### 步骤6：判断金山谷
GV(t) = I(MA5(t-1) < MA10(t-1) 且 MA5(t-1) < MA20(t-1) 且 MA5(t) >= MA10(t) 且 MA5(t) >= MA20(t))

### 步骤7：筛选金山谷样本并统计
设金山谷样本集为S_GV = {t: GV(t) = 1}

上涨概率：
P_up = N(t in S_GV 且 R_next(t) > 0) / N(S_GV) * 100%

平均涨跌幅：
R_mean = (1/N(S_GV)) * Σ R_next(t), t属于S_GV

标准差：
S = sqrt((1/(N-1)) * Σ (R_next(t) - R_mean)^2)

95%置信区间：
R_mean ± 1.96 * S / sqrt(N)
