# 股票问题数学建模 - 完整数学逻辑与建模过程

本文档详细阐述西南财经大学数学建模竞赛中四个股票量化投资问题的数学建模方法。

## 一、基础数据定义

### 1.1 原始数据字段

| 字段名 | 含义 | 示例 |
|--------|------|------|
| ts_code | 股票代码 | 600000.SH, 300001.SZ |
| trade_date | 交易日期 | 20250506 |
| open | 开盘价 | 10.50 |
| high | 最高价 | 11.20 |

### 1.2 基本变量定义

| 变量 | 含义 |
|------|------|
| P(t) | 第t日收盘价 |
| P(t-1) | 第t-1日收盘价 |
| P(t+1) | 第t+1日收盘价 |
| OP(t) | 第t日开盘价 |
| HP(t) | 第t日最高价 |
| LP(t) | 第t日最低价 |

### 1.3 涨跌幅计算公式

涨跌幅(t) = (P(t) - P(t-1)) / P(t-1) * 100%

## 三、问题1：涨停次日走势分析

### 3.1 涨停定义

涨停(t) = 1, 如果 涨跌幅(t) >= 9.9%
         = 0, 否则

即：涨停(t) = I((P(t) - P(t-1))/P(t-1) * 100% >= 9.9%)

### 3.2 次日涨跌幅定义

次日涨跌幅 = (P(t+1) - P(t)) / P(t) * 100%

### 3.3 涨跌幅区间划分

主板区间：

| 区间i | 条件 |
|--------|------|
| 涨停 | 次日涨跌幅 >= 9.9% |
| 涨幅大于7 | 7% < 次日涨跌幅 < 9.9% |
| 5到7 | 5% < 次日涨跌幅 <= 7% |
| 3到5 | 3% < 次日涨跌幅 <= 5% |
| 0到3 | 0% < 次日涨跌幅 <= 3% |
| 负3到0 | -3% < 次日涨跌幅 <= 0% |
| 负5到负3 | -5% < 次日涨跌幅 <= -3% |
| 跌停 | 次日涨跌幅 <= -5% |

创业板区间类似，跌停阈值为-10%。

### 3.4 建模过程

步骤1：数据排序
对每只股票，按日期升序排列

步骤2：计算次日收盘价
P_next(t) = P(t+1) = shift(P, -1)

步骤3：计算次日涨跌幅
R_next(t) = (P_next(t) - P(t)) / P(t) * 100%

步骤4：识别涨停
Z(t) = I(R(t) >= 9.9%)

步骤5：板块分类
B(t) = 板块(ts_code(t))

步骤6：统计概率分布
P(i) = N(t: R_next(t)属于区间i 且 Z(t)=1 且 B(t)=b) / N(t: Z(t)=1 且 B(t)=b) * 100%

## 四、问题2：均线金叉分析

### 4.1 移动平均线(MA)定义

MA(n, t) = (1/n) * ΣP(t-i), i从0到n-1

具体：
MA(5, t) = (P(t) + P(t-1) + P(t-2) + P(t-3) + P(t-4)) / 5
MA(10, t) = (P(t) + P(t-1) + ... + P(t-9)) / 10

### 4.2 金叉定义

金叉(t) = 1, 如果 MA(5, t-1) < MA(10, t-1) 且 MA(5, t) >= MA(10, t)
        = 0, 否则

即：金叉(t) = I(MA5(t-1) < MA10(t-1) 且 MA5(t) >= MA10(t))

### 4.3 建模过程

步骤1：计算MA5
MA5(t) = rolling(P, window=5).mean()

步骤2：计算MA10
MA10(t) = rolling(P, window=10).mean()

步骤3：计算次日涨跌幅
R_next(t) = shift(R(t), -1)

步骤4：计算均线前一时刻值
MA5(t-1) = shift(MA5(t), 1)
MA10(t-1) = shift(MA10(t), 1)

步骤5：判断金叉
G(t) = I(MA5(t-1) < MA10(t-1) 且 MA5(t) >= MA10(t))

步骤6：统计
上涨概率 = N(t: G(t)=1 且 R_next(t)>0) / N(t: G(t)=1) * 100%
平均涨跌幅 = (1/N) * Σ R_next(t), t满足G(t)=1

## 五、问题3：金山谷形态分析

### 5.1 均线定义

MA(5, t) = (P(t) + P(t-1) + P(t-2) + P(t-3) + P(t-4)) / 5
MA(10, t) = (P(t) + P(t-1) + ... + P(t-9)) / 10
MA(20, t) = (P(t) + P(t-1) + ... + P(t-19)) / 20

### 5.2 银山谷定义

银山谷(t) = 1, 如果 MA5(t-1) < MA10(t-1) 且 MA5(t-1) < MA20(t-1) 且 MA5(t) >= MA10(t) 且 MA5(t) >= MA20(t)
           = 0, 否则

### 5.3 金山谷定义

金山谷(t) = 1, 如果 存在k>0使得银山谷(t-k)=1 且 MA5(t) >= MA10(t) 且 MA5(t) >= MA20(t)
          = 0, 否则

### 5.4 金山谷vs银山谷对比

| 形态 | MA5(t-1)与MA10(t-1) | MA5(t-1)与MA20(t-1) | MA5(t)与MA10(t) | MA5(t)与MA20(t) | 时间约束 |
|------|----------------------|----------------------|----------------|----------------|----------|
| 银山谷 | 小于 | 小于 | 大于等于 | 大于等于 | 首次 |
| 金山谷 | 小于 | 小于 | 大于等于 | 大于等于 | 银山谷之后 |

### 5.5 建模过程

与问题2类似，额外增加MA20计算和银山谷/金山谷判断条件。

## 六、问题4：封板时间分析

### 6.1 封板判定

封板(t) = I(HP(t) = P(t))

### 6.2 封板类型分类

秒封 = I(HP(t) = P(t) 且 LP(t) = P(t))
早盘封板 = I(HP(t) = P(t) 且 LP(t) < P(t))
午后尾盘封板 = I(HP(t) > P(t))

### 6.3 建模过程

步骤1：识别涨停
Z(t) = I(R(t) >= 9.9%)

步骤2：计算次日涨跌幅
R_next(t) = shift(R(t), -1)

步骤3：分类封板类型
C(t) = 秒封/早盘封板/午后尾盘封板

步骤4：按类型统计
对每个类型j计算：
- 上涨概率 = N(t: C(t)=j 且 R_next(t)>0) / N(t: C(t)=j) * 100%
- 平均涨跌幅 = (1/Nj) * Σ R_next(t), C(t)=j
- 连板概率 = N(t: C(t)=j 且 R_next(t)>=9.9%) / N(t: C(t)=j) * 100%

## 七、数据处理关键修正

### 7.1 次日数据定义修正

错误方法：先筛选事件，再平移
正确方法：先平移计算全局次日数据，再筛选事件

正确代码：
R_next(t) = shift(R(t), -1)
样本集S = {t: 事件(t)=1}

### 7.2 板块分类修正

错误：使用endswith(".SH")判断
正确：使用代码前3位判断

### 7.3 创业板认定修正

错误：仅判断300开头
正确：包含300和301开头
