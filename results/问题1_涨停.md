# 问题1: 涨停次日走势分析

## 数学逻辑

### 1.1 基本变量定义

| 变量 | 含义 |
|------|------|
| P(t) | 第t日收盘价 |
| P(t+1) | 第t+1日收盘价(次日) |
| HP(t) | 第t日最高价 |
| LP(t) | 第t日最低价 |

### 1.2 板块分类

板块(代码) = 主板，    如果 代码属于 {600,601,603,000,001}开头
           创业板/科创板，如果 代码属于 {300,301}开头

### 1.3 涨停定义

涨停(t) = 1, 如果 (P(t) - P(t-1))/P(t-1) * 100% >= 9.9%
        = 0, 否则

### 1.4 次日涨跌幅

次日涨跌幅 = (P(t+1) - P(t))/P(t) * 100%

### 1.5 涨跌幅区间划分

主板区间划分：

| 区间i | 条件 |
|--------|------|
| 涨停 | 次日涨跌幅 >= 9.9% |
| 涨幅大于7 | 7% < 次日涨跌幅 < 9.9% |
| 5到7 | 5% < 次日涨跌幅 <= 7% |
| 3到5 | 3% < 次日涨跌幅 <= 5% |
| 0到3 | 0% < 次日涨跌幅 <= 3% |
| 负3到0 | -3% < 次日涨跌幅 <= 0% |
| 负5到负3 | -5% < 次日涨跌幅 <= -3% |
| 跌停 | 次日涨跌幅 <= -5% |

创业板区间划分类似，区别在于跌停阈值为-10%。

---

## 建模过程

### 步骤1：数据准备
设原始数据集为D，包含N条记录，每条记录包含(ts_code, trade_date, P, HP, LP, pre_close)

### 步骤2：计算次日收盘价
对每只股票，按日期排序后：
P_next(t) = P(t+1) = shift(P, -1)

### 步骤3：计算次日涨跌幅
R_next(t) = (P_next(t) - P(t))/P(t) * 100%

### 步骤4：识别涨停
Z(t) = 1, 如果 R(t) >= 9.9%，否则 0

### 步骤5：板块分类
B(ts_code) = 主板, 如果 ts_code属于主板代码集
           创业板/科创板, 如果 ts_code属于创业板代码集

### 步骤6：计算概率分布
P(i) = N(i) / N_total * 100%

其中N(i)为第i个区间的样本数，N_total为该板块涨停总样本数。
