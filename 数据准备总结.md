- **竞赛时间**: 2026年5月1日

## 需要准备的数据

### 数据总要求
- 时间范围：近一年（20250501-20260430）
- 范围：全部A股日交易数据
- 剔除：ST股票、退市股票、上市不足一年的新股

---

### 问题1：涨停相关数据

**数据类型**：
- A股日交易数据
- 包含字段：股票代码、日期、开盘价、收盘价、最高价、最低价、涨跌幅等

**需要分析**：
- 主板个股涨停后次日走势概率分布
- 创业板/科创板个股涨停后次日走势概率分布

**具体分组**：
| 涨幅区间 | 主板 | 创业板/科创板 |
|---------|------|--------------|
| 涨停 | ✓ | ✓ |
| >7% | ✓ | - |
| 5~7% | ✓ | ✓ |
| 3~5% | ✓ | ✓ |
| 0~3% | ✓ | ✓ |
| -3~0% | ✓ | ✓ |
| -5~-3% | ✓ | ✓ |
| <-7% | ✓ | - |
| -7~-5% | - | ✓ |
| -10~-7% | - | ✓ |
| <-10% | - | ✓ |
| 跌停 | ✓ | ✓ |

---

### 问题2：金叉相关数据

**数据类型**：
- A股日交易数据
- 需计算均线：5日均线、10日均线

**分析方法**：
- 以5日均线与10日均线金叉为买点
- 计算购买后次日起涨的概率
- 涨跌幅的点估计与区间估计

---

### 问题3：金山谷相关数据

**数据类型**：
- A股日交易数据
- 需计算均线：短期均线、中期均线、长期均线

**分析方法**：
- 识别金山谷形态（金叉后再次金叉形成的三角形）
- 以金山谷出现位置为买入点
- 分析短期（10日左右）上涨概率
- 涨跌幅的点估计与区间估计

---

### 问题4：封板时间相关数据

**数据类型**：
- A股分时数据（某日）
- 日交易数据

**分析方法**：
按四个时间段分组：
| 时间段 | 时段范围 |
|-------|----------|
| 早盘 | 9:35-10:30 |
| 午前 | 10:30-11:30 |
| 午后 | 13:00-14:30 |
| 尾盘 | 14:30-15:00 |

**分析指标**：
- 封板次日起涨的概率
- 涨跌幅的点估计与区间估计
- 次日连板的概率
